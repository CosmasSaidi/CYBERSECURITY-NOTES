## Suggested Learning Path

Follow this order if you're new to security:

1. **Linux Fundamentals** → permissions → environment → path → processes → privileges
2. **Shell Scripting** → substitution → pipes → safe patterns
3. **Reverse Engineering** → workflow → tools
4. **Binary Exploitation** → memory layout
